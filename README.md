# RefleK's - Open Source

## The Repository

This repository is home to the source code for RefleK's. It is developed in the open, and available under the GNU General Public License v3.0.

## RefleK's

RefleK's is a local desktop app for Kovaak's analytics, session tracking, and benchmarking.

Built with a high-performance Go backend and a modern React frontend, RefleK's provides local, real-time analytics and actionable insights for your aim training.

Download the latest release for Windows at [refleks-app.com](https://refleks-app.com).

## Development Container

This repository includes a Visual Studio Code Dev Container configuration.

Use the **Dev Containers: Clone Repository in Container Volume...** command in VS Code to clone the source code into a container volume and spin up a dev container with all dependencies pre-installed.
