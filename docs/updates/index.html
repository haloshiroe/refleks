31 Oct 2025
v0.3.1 - Patch
  • Auto‑update (Windows): One‑click updates now work reliably.
30 Oct 2025
v0.3.0 - Minor release
Preview: Sessions - “Next high score forecast” in the Progress (all) tab predicts number of runs to the next PB, with an optimal pause (minutes) and a confidence estimate.
Preview: Sensitivity analysis - see "Score vs Sensitivity (cm/360)" in Sessions (Overview & Progress all). Supports cm/360, in/360, and CSGO scales; unsupported scales are omitted.
Preview features are early, experimental additions we ship to collect feedback. They are functional but may change as we refine the model and UI. To report feedback, click Support & Feedback in the app or go to support.
  • Auto‑update (Windows): the app now checks for updates on startup and lets you install the latest version with one click. You can also trigger a manual check from Settings → Updates.
  • Benchmarks: new “Analysis” tab with side‑by‑side charts for strengths/weaknesses and rank distribution.
  • Settings: Advanced “SteamID override” (Windows) - if set, RefleK's will use this SteamID for fetching benchmark progress instead of parsing Steam's loginusers.vdf. The environment override REFLEKS_STEAM_ID remains supported.
  • Various UI improvements and bug fixes.
28 Oct 2025
v0.2.0 - Minor release
  • Benchmarks: new “Recom” column to suggest what to play next, with up/down indicator and negative scores when it’s better to switch scenarios.
  • New “Progress (all)” tab in Sessions: see all‑time progress across scenarios with trends.
  • Added time‑of‑day chart to visualize when you train most.
  • Switch between Runs and Sessions view in the top chart and summary.
  • Session length insights: warm‑up estimate and suggested runs for average, consistency, and high‑score goals.
  • Added new preset themes: Nord, Gruvbox, and Solarized Dark.
  • Various UI improvements and bug fixes.
28 Oct 2025
v0.1.1 - Patch
  • Fix: “Support & Feedback” now opens in your default browser instead of the in‑app webview.
28 Oct 2025
v0.1.0 - Minor release
  • Proper app versioning (semantic version, exposed via the app API)
  • Updated settings system with advanced options (theme, traces dir, parse-existing limit, mouse buffer)
  • New Raw Stats tab for scenarios with smart formatting and categories
  • Benchmark progress shown inside the Sessions view for the selected scenario
  • Various UI improvements and bug fixes
25 Oct 2025
v0.0.0 - Initial release
  • Automatic tracking of KovaaK's stats with live updates
  • Session grouping by time gap, with session summaries and recent sessions list
  • Scenario analysis: events over time, accuracy trend, Real Avg TTK, moving averages, and consistency
  • Sessions overview: score/accuracy/TTK chart, findings (best/worst runs), and scenario‑mix radar
  • Benchmarks: explore official lists, favorite benchmarks, and fetch progress via Steam (Windows)
  • Mouse movement capture and replay (Windows), saved locally to your traces folder
  • Launch scenarios in KovaaK's directly from the app
  • Themes and a basic settings page

We follow semantic versioning (MAJOR.MINOR.PATCH). Release entries are marked as Minor release or Patch to help distinguish feature releases from small fixes.

In progress

Features currently under active development.

AI‑powered insights

Get intelligent feedback on your sessions with pattern recognition, strengths and weaknesses analysis, and personalized recommendations on what to practice next.

Mouse movement analysis

Automatic detection of overshoots, undershoots, and inefficient movements. See exactly how optimal your mouse control is and where to improve.

Sensitivity analysis

Analyze how your performance changes with different sensitivities. Compare score, accuracy, and Real Avg TTK across runs in the Sessions tab to dial in your best settings and improve overall performance.

Always improving

RefleK's is under active development - new ideas are explored, refined, and shipped regularly. Expect frequent quality‑of‑life updates, smarter recommendations, and fresh analytics as we iterate with your feedback.
